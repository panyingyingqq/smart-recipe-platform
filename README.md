# 智能食谱平台

基于React + TypeScript + Vite开发的智能食谱管理平台，支持食谱浏览、搜索、上传等功能。

## 项目特性
- 🍳 丰富的食谱库
- 🔍 智能搜索功能  
- 📱 响应式设计，支持移动端
- ➕ 用户可上传自定义食谱
- 🚀 部署在Vercel，全球CDN加速

## 技术栈
- **前端框架**: React 18 + TypeScript 5
- **构建工具**: Vite 6
- **样式框架**: TailwindCSS 3
- **状态管理**: Zustand
- **数据库**: Supabase (PostgreSQL)
- **部署平台**: Vercel

## 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
